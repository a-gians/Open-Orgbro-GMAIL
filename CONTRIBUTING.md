# Contributing

Thanks for helping improve ORGBRO X3 Gmail Printer.

Useful contributions include:

- clearer setup instructions;
- safer handling of local runtime files;
- fixes for Gmail polling or Pub/Sub behavior;
- improvements to text layout and raster rendering;
- compatibility notes for different ORGBRO X3 firmware versions.

Before opening a pull request:

- do not commit OAuth credentials, Gmail tokens, logs, email content, local paths, BLE addresses, or personal sample data;
- use placeholders in documentation and examples;
- keep changes small and reproducible;
- include the command you ran and the operating system you tested on.

If a change depends on local captures or private test data, describe the behavior without attaching sensitive files.
